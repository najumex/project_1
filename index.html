Comandos Git

  • Configuracion
  • Repositorios
  • Ramificacion
  • git init
    Inicializa repositorio
  • git status
    Estado de repositorio
  • git log
    Muestra una lista de todos los commits del repositorio
  • git commit -a -m "<Nombre_Commit>"
  • git commit --amend
    Renombra ultimo commit
  • git clone <URL>
    Clona un repositorio desde una direccion
  • git remote add <Nombre_Remoto> <URL>
    Crea una direccion de referencia a un repositorio remoto
  • git remote rm <Name_Remoto>
    Elimina la direccion de referencia indicada
  • git remote rename <Old_Name> <New_Name>
    Renombra
  • git branch
    Muestra todas las ramas existentes
  • git branch <Nombre_Rama>
    Nueva rama
  • git branch -m <old_name_branch> <new_name_branch>
    Renombrar Rama
  • git checkout <Name_Rama>
    Salta a la rama indicada
  • git branch -d <Name_Rama>
    Elimina rama indicada, solo la elimina si la rama no tiene contenido
  • git tag -a <No.Version> -m "<Mensaje>"
    Este comando crea una etiqueta anotada junto con un mensaje que se almacena con la etiqueta
  • git tag
    Este comando lista las etiquetas en orden alfabetico
  • git tag -l '<Referencia>*'
    Este comando filtra las etiquetas dependiendo de la referencia
  • git push <Nombre_Remoto> --tags
    Este comando envia todas las etiquetas que no esten en el servidor remoto
